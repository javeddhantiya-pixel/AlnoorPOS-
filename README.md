# Alnoor Garden POS — Android APK

## APK Automatic Banegi — GitHub Actions se

### STEP 1: GitHub Account
→ github.com pe free account banao

### STEP 2: New Repository
1. github.com/new
2. Name: **AlnoorPOS**
3. Public select karo
4. Create repository click karo

### STEP 3: Files Upload karo
1. "uploading an existing file" click karo
2. Is ZIP ki saari files drag karo
3. "Commit changes" click karo

### STEP 4: APK Download karo
1. Repository mein "Actions" tab click karo
2. "Build APK" workflow click karo
3. Green tick aane tak wait karo (~5 min)
4. "AlnoorPOS" artifact download karo
5. ZIP extract karo → **app-release-unsigned.apk** milegi

### STEP 5: Tablet pe Install karo
1. APK file WhatsApp se tablet pe bhejo
2. Tablet → Settings → Security → Unknown Sources ON
3. APK tap karo → Install
4. **Alnoor POS** app ready! ✅

## Features
- 🖨️ Bluetooth thermal printer direct print
- ☁️ Firebase cloud sync (8 tablets)
- 📱 Fullscreen app
- ⚡ Auto start on boot
